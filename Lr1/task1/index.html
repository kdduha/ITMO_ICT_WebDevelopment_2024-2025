Условие

Реализовать клиентскую и серверную часть приложения. Клиент отсылает серверу сообщение «Hello, server». Сообщение должно отразиться на стороне сервера. Сервер в ответ отсылает клиенту сообщение «Hello, client». Сообщение должно отобразиться у клиента.

Реализовать с помощью протокола UDP

Используемый стэк

go version go1.22.5 darwin/arm64

Используются только базовые фреймворки и пакеты (net, net\http, log и т.д.)

Запуск

Параметры запуска сервера:

go run task_1/server.go -n <number of requests> -p <port>

Параметры запуска клиента:

go run task_1/client.go -n <number of requests> -p <port> -s <seconds between requests>

Клиент и сервер отправляют друг другу сообщения в течение заданного количества запросов. Если число отправленных запросов у клиента меньше, чем ожидаемых принятых у сервера, то клиент просто отключается. В обратном случае сервер отключится первым и клиент поймает ошибку
